# Woodland Estate & Title — Website

Static multi-page marketing site for Woodland Estate & Title, a Capitol Hill-based title and settlement firm serving Washington DC, Maryland, and Virginia.

## Stack

Static HTML + CSS. No build step. Deploy-ready for Vercel, Netlify, GitHub Pages, or any static host.

## Pages

- `index.html` — Home
- `services.html` — Services for buyers, sellers, and professionals
- `closing-process.html` — Three-phase closing walkthrough
- `code-of-conduct.html` — Values: transparency + accessibility
- `contact.html` — Contact info + Capitol Hill map
- `security.html` — Data security and wire fraud prevention
- `covid-19.html` — Flexible closing options
- `utilities.html` — Utility provider directory (DC, MD, VA)

## Structure
